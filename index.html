🚀 Quick Setup Guide

1. Clone Repository

git clone https://github.com/Nagesh00/cryptopulseai.git
cd cryptopulseai

2. Setup Environment

python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt

3. Configure API Keys

cp config_template.py config.py
# Edit config.py with your API keys

4. Run Application

python app.py
# Visit http://localhost:5000
